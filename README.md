# FreedomForAfrica

Ce projet a pour but de regrouper différentes informations sur les censures de 26 pays démocratiques d'Afrique.

Elle contient une carte interactive de l'Afrique qui à terme comprendra un aperçu de la censure grâce à un code couleur basé sur un système de points à partir de différents types de barrages à la liberté d'internet.
(Obstacle à l'accès, Limites en matière de contenu, Violations des droits des utilisateurs.)

Une Home Page ainsi qu'une page About en construction.

Une page contenant la liste des pays sous forme de carte qui contiendra à terme plus d'informations.

Une page pour chaque pays ainsi que les différentes informations concernant la censure d'internet de celui-ci.


## Prerequisites

Install Node.js version 16.0 or higher

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
